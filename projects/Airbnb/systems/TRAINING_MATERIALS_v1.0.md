## 📋 TRAINING MODULE 1: PROPERTY ACQUISITION SUPPORT

### **Role: Property Search Assistant**
**Responsibilities:** Help with agent communication, property evaluation, documentation

#### **Training Topics:**
1. **Understanding Property Criteria:**
   - Target: 1R/1K apartments (20-30m²)
   - Location: Fukuoka City (Chuo-ku, Hakata-ku, Minami-ku preferred)
   - Rent: ¥80,000-¥120,000/month (target: ¥100,000)
   - **CRITICAL:** Building must allow minpaku
   - Condition: Good, minimal renovation needed
   - Access: Near station (5-10 min walk)

2. **Agent Communication Protocol:**
   - Use prepared email templates (Templates A1-A5)
   - Always copy Saito on all communications
   - Track responses in shared spreadsheet
   - Follow up after 24 hours if no response
   - Schedule calls with responsive agents

3. **Property Evaluation Support:**
   - Help gather property details from agents
   - Apply Property Evaluation Framework (scoring 1-10)
   - Document key information in property tracking sheet
   - Flag any potential issues (noise, condition, location concerns)

4. **Documentation Management:**
   - Save all property documents to shared drive
   - Organize by property ID and date
   - Ensure critical documents are obtained:
     - Floor plans
     - Building rules (minpaku allowance confirmation)
     - Photos (interior and exterior)
     - Rent and fee details

#### **Success Metrics for Property Assistant:**
- **Response Rate:** 90%+ of agent communications responded to within 24h
- **Documentation:** 100% of property details properly documented
- **Evaluation:** Consistent application of evaluation framework
- **Timeline:** Property search completed within 14 days target

### **Training Exercises:**
1. **Exercise 1:** Review 3 sample property listings and apply evaluation criteria
2. **Exercise 2:** Draft follow-up email to agent using template
3. **Exercise 3:** Organize sample property documents in shared drive structure
4. **Exercise 4:** Schedule viewing appointments using calendar system

## 🚀 ONBOARDING PROCESS

### **Step 1: Initial Assessment (Day 1)**
- Review assistant's experience and skills
- Determine which modules are relevant
- Set clear expectations and success metrics
- Establish communication channels and frequency

### **Step 2: System Orientation (Day 1-2)**
- Tour of all systems and tools
- Access setup for shared drives, spreadsheets, dashboards
- Introduction to key templates and checklists
- Security and confidentiality briefing

### **Step 3: Module Training (Day 2-5)**
- Complete relevant training modules
- Practice with training exercises
- Shadow Saito on actual tasks (if possible)
- Gradual increase in responsibility

### **Step 4: Independent Work with Oversight (Day 6-14)**
- Begin handling actual tasks with supervision
- Regular check-ins and feedback sessions
- Quality review of completed work
- Adjustment of responsibilities based on performance

### **Step 5: Full Integration (Day 15+)**
- Independent work with periodic reviews
- Weekly performance assessment
- Continuous improvement based on feedback
- Expansion of responsibilities as proficiency grows

## ⚠️ RISK MANAGEMENT IN TRAINING

### **Common Risks & Mitigation:**
1. **Miscommunication Risk:**
   - **Mitigation:** Clear templates, regular check-ins, written confirmation
   - **Training:** Communication protocol training, template usage

2. **Quality Risk:**
   - **Mitigation:** Checklists, quality control steps, supervision
   - **Training:** Quality standards training, checklist usage

3. **Timeline Risk:**
   - **Mitigation:** Clear deadlines, progress tracking, escalation process
   - **Training:** Timeline management training, priority setting

4. **Cost Risk:**
   - **Mitigation:** Budget limits, approval process, tracking
   - **Training:** Budget awareness training, expense tracking

5. **Compliance Risk:**
   - **Mitigation:** Clear guidelines, supervision, verification
   - **Training:** Regulatory requirement training, compliance checklist

### **Escalation Protocol:**
1. **Level 1:** Assistant handles independently using guidelines
2. **Level 2:** Assistant consults with Saito for guidance
3. **Level 3:** Saito takes over handling directly
4. **Emergency:** Immediate contact for critical issues

**Escalation Triggers:**
- Financial decisions above delegated limit
- Guest complaints or negative reviews
- Regulatory or compliance issues
- Vendor disputes or service failures
- Any situation outside training guidelines

- **Assistant Role:** General support across all areas

- **Training Focus:** Management and coordination skills
- **Supervision:** Strategic oversight only
- **Goal:** Self-managing team structure

- **Initial:** Comprehensive training for general support

- **Mature:** Management training for team leadership
- **Continuous:** Regular updates for system improvements
