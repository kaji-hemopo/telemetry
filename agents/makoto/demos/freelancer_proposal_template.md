## What Makes Freelancer.com Proposals Work

1. **Single focus** — one clear skill demonstration, not a services list
2. **Start price explicit** — "$150 flat for standard scraper" beats "let's discuss"
3. **Demo link first** — a live URL proves capability instantly
4. **48h response** — Freelancer.com moves fast; follow up at 24h mark
5. **Account history** — incomplete profile = low response. Push empty profile first.
