## 成功させるコツ / What Makes It Work

1. **Copy exact job language** — verbatim phrases from listing show you actually read it
2. **Demo links first** — "here's a working example of X" = instant credibility over profile text
3. **Short + specific** — 200-300 chars, 3 bullet max, no fluff
4. **Bilingual tag** — "日本語対応OK" in JP version is a differentiator vs EN-only competitors
5. **Follow up in 48h** — if no reply, send 1 follow-up with minor update to proposal
