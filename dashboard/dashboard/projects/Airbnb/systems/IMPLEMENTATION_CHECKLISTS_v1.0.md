### **Phase 1B: Property Search & Evaluation (Day 3-7)**
**Objective:** Identify 3-5 promising properties for viewing

#### **Property Criteria:**
- [ ] **Location:** Fukuoka City, preferably Chuo-ku, Hakata-ku, or Minami-ku
- [ ] **Type:** 1R, 1K, or 1DK (20-30m² ideal)
- [ ] **Rent:** ¥80,000-¥120,000/month (target: ¥100,000)
- [ ] **Building:** Must allow minpaku (CRITICAL)
- [ ] **Condition:** Good condition, minimal renovation needed
- [ ] **Access:** Near station (5-10 min walk), amenities nearby
- [ ] **Layout:** Separate bathroom, kitchenette possible

#### **Evaluation Process:**
- [ ] Receive listings from agents (target: 10-15 initial options)
- [ ] Apply Property Evaluation Framework (from Property Search System)
- [ ] Score each property (1-10 scale)
- [ ] Shortlist top 5 properties for agent discussion
- [ ] Schedule viewings for top 3 properties

#### **Viewing Preparation:**
- [ ] Create viewing checklist (bring: measuring tape, phone for photos, notebook)
- [ ] Prepare questions for each property:
  1. Confirm building allows minpaku (get in writing if possible)
  2. Check noise insulation between units
  3. Verify internet availability and speed
  4. Check water pressure and heating
  5. Assess furniture fit and layout
  6. Note any needed repairs or improvements

### **Phase 1C: Property Selection & Negotiation (Day 8-14)**
**Objective:** Secure best property with favorable terms

#### **Selection Criteria:**
- [ ] **Minpaku Allowance:** Confirmed in writing from building management
- [ ] **Landlord Openness:** Willing to give written permission
- [ ] **Rent Negotiation:** Target 10-15% below asking if possible
- [ ] **Deposit/Key Money:** Standard 2 months rent + 1 month key money
- [ ] **Lease Terms:** 2-year minimum, minpaku clause included
- [ ] **Condition:** Minimal renovation needed (<¥50,000)

#### **Negotiation Strategy:**
- [ ] Use agent as intermediary for initial negotiation
- [ ] Highlight responsible operation and insurance coverage
- [ ] Offer higher deposit if needed for landlord comfort
- [ ] Request minpaku clause in lease agreement
- [ ] Secure written landlord permission before signing

#### **Final Decision:**
- [ ] Compare top 2 properties using decision matrix
- [ ] Consider: Location score, rent, condition, landlord cooperation
- [ ] Make selection based on overall best fit for minpaku
- [ ] Proceed to lease signing with selected property

## ⚠️ RISK MITIGATION CHECKLISTS

### **Risk 1: Landlord Permission Denied**
**Mitigation Steps:**
- [ ] Verify building allows minpaku BEFORE property search
- [ ] Work with agents experienced in minpaku properties
- [ ] Offer higher deposit or rent premium if needed
- [ ] Secure written permission before lease signing
- [ ] Have backup properties identified

### **Risk 2: Licensing Delays**
**Mitigation Steps:**
- [ ] Use consultant to ensure document accuracy
- [ ] Submit complete application (no missing documents)
- [ ] Follow up weekly with city office
- [ ] Have contingency plan for delayed launch
- [ ] Budget for longer approval timeline (8 weeks vs 6)

### **Risk 3: Low Occupancy**
**Mitigation Steps:**
- [ ] Use dynamic pricing to attract early bookings
- [ ] Create compelling listing with professional photos
- [ ] Offer introductory pricing for first month
- [ ] Optimize listing based on early feedback
- [ ] Monitor competitor pricing and adjust accordingly

### **Risk 4: Guest Issues**
**Mitigation Steps:**
- [ ] Clear house rules and communication
- [ ] 24/7 guest support availability
- [ ] Thorough cleaning and quality control
- [ ] Quick response to any issues
- [ ] Learn and improve from each guest experience

### **Risk 5: Operational Overload**
**Mitigation Steps:**
- [ ] Systematize all repeatable processes
- [ ] Use automation for messaging and scheduling
- [ ] Establish reliable service providers
- [ ] Monitor time spent and optimize continuously
- [ ] Target: <30 minutes/day management from Month 2
